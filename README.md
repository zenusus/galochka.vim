# galochka vim plugin
крутой плагин для вставки галочек на положение курсора для vim <br>
для вставки галочек напишите `:galochka 5` (или другое кол-во)

## как ставить
используй любой менеджер плагинов (например vim-plug)<br>
вставь себе в .vimrc где плагины:<br>
```
call plug#begin()
Plug 'zenusus/galochka.vim'
call plug#end()
```
